# My AI Clone - User Guide

## Quick Start Guide

### 1. First Time Setup

When you first open the app, you'll see the home screen with three main options:

#### Get Your API Key
1. Visit https://platform.openai.com
2. Create an account or sign in
3. Go to "API Keys" in your account settings
4. Click "Create new secret key"
5. Copy the key (it starts with "sk-")
6. **Important**: Save this key somewhere safe - you won't be able to see it again!

#### Configure the App
1. Tap the **Settings** icon (‚öôÔ∏è) in the top right
2. Paste your API key in the "OpenAI API Key" field
3. Choose your AI model:
   - **GPT-3.5 Turbo**: Faster and cheaper (recommended for testing)
   - **GPT-4**: More accurate but slower and more expensive
4. Tap **Save Settings**

### 2. Import Your Chat History

#### Export from WhatsApp
1. Open WhatsApp on your phone
2. Open any chat (individual or group)
3. Tap the three dots (‚ãÆ) in the top right
4. Select **More** ‚Üí **Export chat**
5. Choose **Without media** (important!)
6. Save the .txt file to your device

#### Import into the App
1. In My AI Clone, tap **Import Chats**
2. Enter your name exactly as it appears in WhatsApp
   - Example: If your messages show "John", enter "John"
   - If they show "You", enter "You"
3. Tap **Select Chat File**
4. Choose the .txt file you exported
5. Wait for the import to complete
6. You'll see a success message with the number of messages imported

**Tips for Better Results:**
- Import multiple chats to give the AI more data
- Include chats with different people to capture your various communication styles
- The more messages you import, the better the AI will understand your style

### 3. Test Your AI Clone

1. Tap **Test AI Clone** from the home screen
2. Type a message as if someone is asking you something
3. Tap the send button
4. Wait for the AI to generate a response
5. See how the AI responds in your style!

**Example Conversations to Try:**
- "What are you doing this weekend?"
- "Can you help me with something?"
- "Want to grab coffee?"
- "How was your day?"

### 4. Using Smart Replies (Future Feature)

Currently, Smart Replies shows demo data. In the future:

3. Tap a suggestion to copy it

## Troubleshooting

### "Please set your API key in Settings"
- You haven't entered an API key yet
- Go to Settings and add your OpenAI API key

### "Error importing: No messages found"
- The chat file format might be incorrect
- Make sure you exported "Without media"
- Check that your name is spelled exactly as in the chat

### "Error generating response"
- Check your internet connection
- Verify your API key is correct
- Ensure you have credits in your OpenAI account
- Try using GPT-3.5 instead of GPT-4

### Import shows 0 messages
- Your name doesn't match the name in the chat
- The chat file might be corrupted
- Try exporting the chat again

### AI responses don't sound like me
- Import more chat data (aim for 500+ messages)
- Include chats with different people
- Make sure the imported chats are recent

## Privacy & Security

### Your Data
- All chat data is stored **locally** on your device
- Messages are only sent to OpenAI when generating responses
- No data is stored on any other servers
- You can delete all data anytime from Settings

### API Key Security
- Your API key is stored securely on your device
- Never share your API key with anyone
- If compromised, delete it from OpenAI and create a new one

### Platform Policies
- This app does NOT automatically send messages
- It only generates suggestions that you send manually
- This ensures compliance with WhatsApp/Instagram Terms of Service
- Automatic sending could result in account bans

## Frequently Asked Questions

**Q: Can I use this without an API key?**  
A: No, you need an OpenAI API key to generate AI responses.

**Q: Is my data safe?**  
A: Yes, all data stays on your device except when sent to OpenAI for generating responses.

**Q: Can I import Instagram chats?**  
A: Not yet, but it's planned for a future update.

**Q: Will this auto-reply to my messages?**  
A: No, it only generates suggestions. You must manually send them.

**Q: How many chats should I import?**  
A: The more the better! Aim for at least 500-1000 messages for good results.

**Q: Can I delete my imported data?**  
A: Yes, go to Settings ‚Üí Data Management ‚Üí Delete All Data.

**Q: Does this work offline?**  
A: No, it needs internet to connect to OpenAI's API.

**Q: Can multiple people use this app?**  
A: Yes, but each person needs their own API key and should import their own chats.
